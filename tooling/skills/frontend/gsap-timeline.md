> **License**: MIT
> **Use when**: Sequencing multiple animations, creating animation timelines, controlling playback

## Overview
Teaches the agent how to create and control GSAP timelines for orchestrating complex animation sequences with labels, nesting, position parameters, and playback control.

## Key Topics
- Timeline creation and configuration
- Position parameters (absolute, relative, labels)
- Nesting timelines
- Playback control (play, pause, reverse, seek)
- Timeline callbacks and events
