# GSAP Utility Functions

> **Source**: `tooling/sources/github/gsap-skills/skills/gsap-utils/SKILL.md`
> **License**: MIT
> **Use when**: Using GSAP helper functions for math, arrays, and value manipulation

## Overview
Teaches the agent how to use GSAP's utility functions for common operations: value clamping, range mapping, random values, array conversion, and other helper methods.

## Key Topics
- gsap.utils.clamp() for value bounds
- gsap.utils.mapRange() for value mapping
- gsap.utils.random() for randomization
- gsap.utils.toArray() for selector conversion
- gsap.utils.wrap() and wrapYoyo()
- gsap.utils.interpolate()
