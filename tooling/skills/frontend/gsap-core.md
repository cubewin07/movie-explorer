# GSAP Core Animation

> **Source**: `tooling/sources/github/gsap-skills/skills/gsap-core/SKILL.md`
> **License**: MIT
> **Use when**: Creating web animations with GSAP tweens, easing, staggers, defaults, or matchMedia

## Overview
Teaches the agent how to use GSAP's core animation API correctly: `gsap.to()`, `gsap.from()`, `gsap.fromTo()`, easing functions, stagger configurations, and responsive animations with `gsap.matchMedia()`.

## Key Topics
- Tween creation (to, from, fromTo, set)
- Easing functions and custom eases
- Stagger animations
- gsap.defaults() configuration
- gsap.matchMedia() for responsive animations
- Callback functions (onComplete, onUpdate, etc.)
