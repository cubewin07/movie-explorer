# GSAP Plugins

> **Source**: `tooling/sources/github/gsap-skills/skills/gsap-plugins/SKILL.md`
> **License**: MIT
> **Use when**: Using GSAP plugins like ScrollTo, Flip, Draggable, SplitText, MorphSVG, CustomEase, GSDevTools

## Overview
Teaches the agent how to use GSAP's ecosystem of plugins for advanced animation effects: page scrolling, layout animations, drag interactions, text splitting, SVG morphing, and debugging.

## Key Topics
- ScrollToPlugin (smooth page scrolling)
- Flip plugin (layout transition animations)
- Draggable (drag-and-drop interactions)
- SplitText (text animation by character/word/line)
- MorphSVG (SVG shape morphing)
- CustomEase and GSDevTools
