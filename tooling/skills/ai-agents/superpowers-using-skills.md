## Overview
Teaches the agent how to discover, load, and compose skills for a given task. Covers skill selection strategy, priority management, and multi-skill workflows.

## Key Topics
- Skill discovery and selection
- Loading strategy (selective, not exhaustive)
- Skill composition and chaining
- Priority and conflict resolution
- Reporting which skills were used
