# Brainstorming & Requirements Refinement

> **Source**: `tooling/sources/github/superpowers/skills/brainstorming/SKILL.md`
> **License**: MIT
> **Use when**: Starting a new feature, clarifying requirements, exploring solutions

## Overview
Forces the agent into a Socratic refinement process before writing any code. The agent interviews the user to surface assumptions, edge cases, and design tradeoffs.

## Key Topics
- Requirement elicitation through questioning
- Assumption surfacing
- Edge case identification
- Solution space exploration
- Visual companion for brainstorming
